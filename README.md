# sysadmin

[![npm version](https://badge.fury.io/js/sysadmin.svg)](https://www.npmjs.com/package/sysadmin)

System Administration Web UI


### Development

    # start the webpack client-side compiler process
    make dev
    # start the typescript server-side compiler process
    node_modules/.bin/tsc -w
    # start the server
    make server
